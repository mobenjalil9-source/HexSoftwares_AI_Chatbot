# HexSoftwares_AI_Chatbot
AI Student Assistant Chatbot using HTML, CSS, JavaScript and Groq API

## Description
This project is an AI-powered chatbot developed using HTML, CSS, JavaScript, and the Groq API (Llama 3.1). It answers questions related to Artificial Intelligence, Machine Learning, Python, NLP, Data Science, and Programming.

## Features
- AI-powered responses
- Clean and responsive UI
- Dark mode
- Chat history
- Typing animation
- Copy response
- Clear chat

## Technologies Used
- HTML
- CSS
- JavaScript
- Groq API
- Llama 3.1

## Setup
1. Get a Groq API key.
2. Replace `YOUR_GROQ_API_KEY` in `index.html` with your own key.
3. Open `index.html` in your browser.
